Front end

Hoy en día, la programación es un ámbito que no para de crecer, desarrollar páginas web, crear aplicaciones y programas que faciliten nuestro día a día o mejoren la experiencia del usuario, son funciones cruciales que desempeñan los desarrolladores. Los términos de frontend y backend, tienen cualidades únicas y distintivas, actúan como unidades que por separado, interactúan entre sí para asegurar la funcionalidad de un sitio

Back end

El backend es la parte del desarrollo web que se encarga de que toda la lógica de una página web funcione. Se trata del conjunto de acciones que pasan en una web pero que no vemos como, por ejemplo, la comunicación con el servidor
